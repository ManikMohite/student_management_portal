# 🎓 Student Management Portal

A web-based **Student Management Portal** built using **Django** that allows administrators to manage student records, courses, marks, and user accounts efficiently. The application provides a simple, responsive interface for performing CRUD operations and maintaining academic data securely.

---

## 🚀 Features

* 👨‍🎓 Student Registration and Login
* 🔐 Secure Authentication System
* 📋 Student Profile Management
* ➕ Add New Students
* ✏️ Update Student Details
* ❌ Delete Student Records
* 📊 Manage Student Marks
* 🔍 Search and Filter Students
* 📚 Course Management
* 📱 Responsive User Interface
* 🛡️ Django Admin Panel
* 💾 SQLite Database Integration

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/ManikMohite/student_management_portal.git
```

### Navigate to the Project

```bash
cd student_management_portal
```

### Create a Virtual Environment

```bash
python -m venv venv
```

### Activate the Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

### Create Superuser

```bash
python manage.py createsuperuser
```

### Run the Development Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
